Video Example

This streams a video from ROS.

To run on the ROS side, run the following commands:

  • roscore
  • roslaunch gscam videofile.launch FILENAME:=/vagrant/videos/bigbuck.mp4
  • rosrun web_video_server web_video_server

The first command spins up ROS, the second command will publish the video file "bigbuck" (which is a horrible fucking movie that I hate, SORRY) and the third command will expose said video on port 8080.

Then, the tag below should have its src set as the image output, and the video should automatially play. Currently there's no way to control the video, still working on that.
